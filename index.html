Features

⚡ Direct integration with the official Minecraft launcher
🧩 Auto-detects installed Minecraft versions
🛠 Simple mod profile switching system
🚀 Fast startup, minimal overhead

How It Works

Nebula Launcher does NOT replace Minecraft. It acts as a control layer that opens and configures the official Mojang launcher automatically.
When you launch a profile, Nebula:

1. Reads your Minecraft installation
2. Applies selected mod configuration
3. Opens the vanilla launcher with arguments
4. Lets Minecraft handle the rest
