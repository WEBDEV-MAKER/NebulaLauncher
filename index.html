What is Nebula Launcher?

Nebula Launcher is a lightweight control layer for Minecraft that works with the official Mojang launcher.

It does NOT replace Minecraft — it enhances how you launch it.

✔ Detects installed Minecraft launcher
✔ Launches vanilla Minecraft automatically
✔ Prepares future mod profiles
✔ Minimal performance impact

How It Works

1. Install Nebula Launcher
2. Open the app
3. Select version or profile
4. Click Play → launches official Minecraft launcher
